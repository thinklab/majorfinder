About the Data & Process

This dashboard provides insights into college major outcomes using official U.S. government data.

Data Sources

  • College Scorecard: Primary source for earnings, debt, and repayment data.
  • IPEDS: Supplemental institutional information.

How It Is Processed

  1. Cleaning: Raw data is filtered for relevance and CIP codes (major identifiers) are normalized.
  2. Merging: Major-specific data is combined with institution-wide metrics.
  3. Aggregation: We calculate distributions of earnings across institutions and compute median debt and repayment rates for each major.
  4. Visualization: The processed data is served as a JSON file and rendered using Plotly.js.

Note: Data represents cohorts from recent years as provided by the Department of Education.
